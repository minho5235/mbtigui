## 📝 프로젝트 소개 (Introduction)
이 프로젝트는 사용자가 간단한 질문에 답하며 자신의 성향(E/I, S/N, T/F, J/P)을 파악할 수 있는 **인터랙티브 테스트 프로그램**입니다.
기존의 복잡한 웹 기반 테스트 대신, **Python**만으로 가볍게 실행할 수 있는 데스크톱 애플리케이션(GUI)으로 구현했습니다.

## ✨ 주요 기능 (Features)
* **GUI 인터페이스**: `tkinter` 라이브러리를 사용하여 직관적인 버튼 클릭 방식 구현.
* **실시간 진행률 표시**: 현재 몇 번째 문제를 풀고 있는지 시각적으로 확인 가능.
* **점수 계산 알고리즘**: 사용자 응답에 따라 4가지 지표(EI, SN, TF, JP) 점수를 누적하여 최종 결과 도출.
* **결과 분석**: 16가지 MBTI 유형에 따른 개발자 성향 코멘트 제공.

## 🛠 기술 스택 (Tech Stack)
* **Language**: Python 3.x
* **GUI Library**: Tkinter (Python 내장 라이브러리)
* **Tools**: VS Code, Git
